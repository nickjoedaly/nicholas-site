Ct
Composition Tutor
Guided diagnostic tool for compositional problem-solving — identify what's wrong with a passage and discover how to fix it
Free MuseScore 4 v1.0
Select a passage that isn't working, then walk through a guided decision tree that helps you name the problem. Covers five domains — harmony, melody, texture, rhythm, and form — with branching questions that narrow from "something feels off" to a specific diagnostic. Each endpoint provides a reframing of the issue, concrete solution strategies, recommended reading from the literature (Adler, Piston, Rimsky-Korsakov), and a "Discuss with Claude AI" button that sends your full diagnostic context to Claude for deeper conversation.
Harmony diagnostics Melody analysis Texture / spacing Rhythm / pacing Form / proportion Claude AI integration Solution spaces Reading references

Installation
Download CompositionTutor.qml from GitHub or the button above

Select a passage in your score, then run Plugins > Composition Tutor

Fs
Form Scaffold
Generate complete formal structures with labeled sections, rehearsal marks, tempo changes, and key signatures
Free MuseScore 4 v3.2
Choose from eight musical form types — Sonata, Rondo, Theme & Variations, Ternary/Minuet & Trio, Binary, Fugue, Concerto, and Song Form — and generate a complete structural scaffold directly into your score. Each form comes with historical presets (Classical Mozart, Romantic Beethoven, etc.), configurable measure counts per section, and optional tempo, time signature, and key changes at structural boundaries. Outputs rehearsal marks, staff text labels, and a form map summary. One-step undo removes the entire scaffold.
8 form types Historical presets Sonata form Rondo Fugue Theme & variations Tempo / key / time sig Rehearsal marks Form map output

Installation & supported forms +
Installation
Download FormScaffold5.qml from GitHub or the button above
Copy to ~/Documents/MuseScore4/Plugins/
Go to Home > Plugins and enable "Form Scaffold". Restart if needed.
Open a score, then run Plugins > Composition > Form Scaffold. Cmd-Z undoes the entire operation.
