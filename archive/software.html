Analyzes selected passages and generates Roman numeral chord labels, identifying inversions, secondary dominants, and common non-chord tones. Supports major, minor, and modal contexts with configurable tolerance for ambiguity.

Installation & changelog +
Installation
Download harmonic-analyzer-v2.0.qml from the button above or from GitHub
In MuseScore 4, go to Plugins > Plugin Manager
Click "Add plugin" and select the downloaded .qml file
Enable the plugin and restart MuseScore. Access from Plugins > Harmonic Analyzer

Rh
Rhythm Transformer
Algorithmic rhythm transformations — augmentation, diminution, retrograde, and permutation
Free MuseScore 4
Select a passage and apply rhythmic transformations: augmentation (double durations), diminution (halve durations), retrograde (reverse order), rotation (shift starting beat), and additive permutation. Works on single voices or across multiple staves simultaneously. A compositional tool as much as a notation utility.
